# 回复生成器 (Love Chatbot)

A personal SPA that helps generate chat reply options.

## Quick Start

```bash
npm install
npm run dev
```

## Env Vars

Create a `.env.local` file in the project root:

```
VITE_GEMINI_API_KEY=your_key_here
```

## Security Note

The Gemini API key is bundled client-side by Vite. This is intentional — this tool is for personal use only. Do not deploy it publicly or share the built output.
